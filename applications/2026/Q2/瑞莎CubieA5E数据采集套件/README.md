本项目拟购置瑞莎 Radxa Cubie A5E 迷你主板，选用 A527 / 1GB 商业级版本。该开发板在本项目中定位为双网口数据采集终端，用于雷达与心音、心电同步采集系统的边缘接入、数据缓存和控制协同。A5E 具备双千兆以太网口，可将雷达数据网络与实验室局域网或上位机网络分离，降低采集链路中的带宽争用和网络配置冲突；板载 M.2 2230 NVMe SSD 插槽可扩展本地高速存储，适合连续采集时缓存原始数据和实验日志；内置 RISC-V MCU 可承担实时触发、同步控制和状态监测任务，使 Linux 主系统负责文件、网络和采集服务，实时控制由 MCU 分担，形成更稳定的实验室采集终端架构。

## 采购理由

- 双千兆网口适合把雷达数据网络与上位机/实验室网络分离，提高多设备同步采集时的链路稳定性。
- M.2 2230 NVMe SSD 插槽可为连续采集提供本地高速缓存，避免原始数据和日志只依赖 SD 卡写入。
- A527 8 核 Linux 平台适合运行采集守护进程、时间戳记录、轻量质量检查和数据转发服务。
- 内置 RISC-V MCU 可用于建设采集系统中的触发、同步、低延迟控制和硬件状态监测能力。
- 1GB 商业级版本价格明确，预算低，适合作为实验室双网口数据采集终端的入门样机。
