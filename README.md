各位同学，实验室关于研修的激励政策即日起废止，周知！激励政策起到了一定的成果激励作用，但引起的学术争议较大，甚至助长了后果严重的功利和交易行为，坏了实验室风气和学缘关系，导师团队无义务为各位毕业提供激励！！关于实验室的管理导师团队将认真思考，重新规范，当前先按以下。
1.研修工作应围绕团队学术主题方向，并能够取得推进性进展。
2.积极挑战难题，主动承担学术挑战，为实验室科研做出实质贡献。
3.在实验室取得或取得与团队相关的成果归属于团队不得私自处置。
4.开展工作必须详实，可重现，保留所有资料，有创新贡献者可署名。
5.与导师共同设计，经导师同意的成果方可署名，原则上不再支持硕士发表论文。 
6.取得推动团队项目落地有实质价值成果团队将制定按项目的激励政策。
7.已取得的成果大家须在毕业前提供全部资料和演示验证，如不能复现团队将行使撤回权力，存在虚假的学位论文将上报暂缓毕业或取消答辩环节。
