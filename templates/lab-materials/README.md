# 团队实验材料购置和经费使用论证书模板

此目录是申报模板，不是实际申报项目，不参与自动统计。

实际申报项目的 `申报书.docx` 必须严格按照 [申报书.docx](./申报书.docx) 的提纲撰写：

- 标题: 团队实验材料购置和经费使用论证书
- 项目名称: `Xx购买/经费需求`
- 摘要: 购买内容、指标简介、目标、经费和结果
- 一、需求分析
- 二、目的用途
- 三、实现目标
- 经费数量
- 四、取得效果或结果
- 五、其他
- 导师签字和日期

实际申报项目的图片、价格、订单截图、复盘记录等放在该项目自己的 `README.md` 和 `assets/` 中。

原始通知内容如需保留，可放入实际项目 README 的“文案资料”部分，不应替代模板说明。
